## What is Coords?

Coords is a mini shell application that provides:
- Real-time cursor position tracking with an overlay window
- Cursor movement automation
- Key binding to mouse buttons
- Double-tap key repetition
- Window positioning control

## Features

✨ **Live Coordinates** - Always see your cursor position  
🎯 **Cursor Control** - Move cursor to exact coordinates  
⌨️ **Key Bindings** - Bind keys to mouse buttons  
🔄 **Double Tap** - Repeat keys with custom delays  
🪟 **Window Management** - Lock or follow cursor position
